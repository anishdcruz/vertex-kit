# Vertex kit

A fresh UI-Kit for web application, made with Vue.js 2.0

**Note:**

> This project is a work in progress, so things are expected to be broken :)

## Installation

```
$ npm install vertex-kit
```

## Quick start

```
import Vue from 'vue'
import VertexKit from 'vertex-kit'

Vue.use(VertexKit)
```

## Components

- Grid: Row, Col
- Panel
- Icon
- Button
- ButtonGroup
- more...
